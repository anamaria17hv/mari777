# Buscador de Menú del Día en Restaurantes

## Descripción del Proyecto
Este proyecto consiste en una aplicación web que permite a los trabajadores de oficina encontrar fácilmente restaurantes cercanos que ofrezcan el menú del día. 

Los restaurantes pueden registrar sus menús diarios con los precios para que los usuarios puedan comparar opciones rápidamente.

## Problema
Los trabajadores de oficina pierden tiempo buscando restaurantes donde almorzar o desayunar.

## Solución
El sistema permite que los restaurantes publiquen su menú del día para que los usuarios lo consulten desde una página web.

## Funciones principales
- Registro de usuarios
- Registro de restaurantes
- Publicación del menú del día
- Búsqueda de restaurantes
- Filtrar por precio
- Ver menú con precios

## Tecnologías utilizadas
- HTML
- CSS


## Cómo ejecutar el proyecto
1. Descargar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo index.html en el navegador.
